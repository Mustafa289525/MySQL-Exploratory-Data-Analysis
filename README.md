## MySQL Exploratory Data Analysis Project

## 📌 Overview
This project focuses on analyzing a dataset using SQL to discover patterns, trends, and insights.

## 🛠 Tools Used
- MySQL, MySQL Workbench

## 🔍 Analysis Performed
- Explored dataset structure  
- Used GROUP BY and aggregate functions  
- Analyzed trends and patterns  
- Identified key insights

## 📊 Key Insights
- Some companies laid off 100% of their workforce, indicating shutdowns or severe financial issues  
- Layoffs were concentrated in specific countries, showing uneven global impact  
- Yearly trends reveal fluctuations in layoffs, reflecting changing economic conditions  
- Monthly analysis shows spikes in layoffs during certain periods  
- Well-funded companies were also involved in large layoffs  
- Rolling totals highlight the cumulative increase in job losses over time
